# 📋 Инструкция по настройке HR-викторины с Google Таблицами

## Что было добавлено:
1. ✅ **Загрузка вопросов из Google Таблицы** — все вопросы хранятся в таблице и загружаются динамически
2. ✅ **Общий рейтинг для всех игроков** — результаты сохраняются в таблицу и видны всем со всех устройств
3. ✅ **Админ-панель для добавления вопросов** — можно добавлять новые вопросы прямо через интерфейс

---

## 🔧 Шаг 1: Создайте Google Таблицу

1. Перейдите на [sheets.google.com](https://sheets.google.com)
2. Создайте новую таблицу (например, "STH HR Quiz")
3. **Создайте первый лист** и переименуйте его в `Questions`
4. **Создайте второй лист** и переименуйте его в `Leaderboard`

### Настройка листа Questions:
В первой строке создайте заголовки:
| A | B | C | D | E | F | G | H | I | J | K |
|---|---|---|---|---|---|---|---|---|---|---|
| Theme | QuestionText | Option1 | Option2 | Option3 | Option4 | CorrectOptionIndex | Feedback1 | Feedback2 | Feedback3 | Feedback4 |

Пример данных:
```
💰 Зарплата | Какой минимальный оклад? | 50000 | 60000 | 70000 | 80000 | 2 | Неверно | Почти | Верно! | Слишком много
⏰ График | Сколько часов в день? | 6 | 8 | 10 | 12 | 1 | Мало | Норма | Много | Очень много
```

### Настройка листа Leaderboard:
В первой строке создайте заголовки:
| A | B | C | D |
|---|---|---|---|
| name | totalPoints | gamesCount | bestScore |

---

## 🔧 Шаг 2: Настройте Google Apps Script

1. В вашей Google Таблице нажмите **Расширения** → **Apps Script**
2. Удалите весь код в редакторе
3. **Скопируйте содержимое файла `GoogleAppsScript.js`** из этого репозитория и вставьте в редактор
4. Нажмите **💾 Сохранить** (назовите проект, например, "HR Quiz Backend")

### Развёртывание веб-приложения:
1. Нажмите кнопку **"Развернуть"** (Deploy) → **"Новое развёртывание"**
2. Выберите тип: **"Веб-приложение"**
3. Описание: "v1"
4. **Выполнять от имени:** "От имени пользователя, выполняющего доступ к веб-приложению" (Me)
5. **У кого есть доступ:** **"Все пользователи"** (Anyone) ⚠️ Это важно!
6. Нажмите **"Развернуть"**
7. Скопируйте **URL веб-приложения** (выглядит как `https://script.google.com/macros/s/.../exec`)

---

## 🔧 Шаг 3: Обновите index.html

1. Откройте файл `index.html`
2. Найдите строки 56-57:
```javascript
const SHEET_ID = "1AIGQFAF4S8EJyfbv7UysM-CWpXMCxhiIM5bkAwL0OS0";
const WEBAPP_URL = "https://script.google.com/macros/s/.../exec";
```
3. **Замените `WEBAPP_URL`** на URL из шага 2
4. **Замените `SHEET_ID`** на ID вашей таблицы (возьмите из URL таблицы между `/d/` и `/edit`)

Пример URL таблицы: `https://docs.google.com/spreadsheets/d/1ABC123xyz/edit`
→ ID таблицы: `1ABC123xyz`

## 🔧 Шаг 4: Проверка работы

### Запустите викторину:
1. Откройте `index.html` в браузере
2. На экране входа вы увидите:
   - Поле для ввода ФИО
   - Кнопку "🚀 Начать викторину"
   - Кнопку "⚙️ Добавить вопрос"
   - Таблицу рейтинга (пока пустую)

### Добавьте первый вопрос через админ-панель:
1. Нажмите **"⚙️ Добавить вопрос"**
2. Выберите тему (или создайте новую в таблице)
3. Заполните:
   - Текст вопроса
   - 4 варианта ответа
   - Номер правильного ответа (0-3)
4. Нажмите **"💾 Сохранить вопрос"**
5. Вопрос появится в Google Таблице!

### Сыграйте в викторину:
1. Введите ФИО и нажмите "Начать викторину"
2. Ответьте на вопросы
3. После завершения игры ваш результат сохранится в общем рейтинге

### Проверьте общий рейтинг:
1. Откройте викторину **в другом браузере или на другом устройстве**
2. Вы увидите тот же рейтинг!
3. Все игроки видят общую таблицу лидеров

## 📊 Формат вопросов в таблице

| Column | Name | Description | Example |
|--------|------|-------------|---------|
| A | Theme | Тема вопроса | 💰 Зарплата |
| B | QuestionText | Текст вопроса | Какой оклад у менеджера? |

| K | Feedback4 | Фидбек для варианта 3 | Слишком много |

## 🎮 Как это работает

### Вопросы:
- При запуске игра загружает все вопросы из Google Таблицы
- Если таблица недоступна — используются встроенные тестовые вопросы
- Каждый раунд выбирается по 1 случайному вопросу из каждой темы

### Рейтинг:
- После каждой игры результат отправляется в Google Таблицу
- Таблица обновляется в реальном времени
- Все игроки видят общий рейтинг

### Добавление вопросов:
- Через админ-панель можно добавлять новые вопросы
- Они сразу появляются в таблице и становятся доступны всем игрокам

---

## ⚠️ Важные замечания

1. **Доступ к таблице:** Убедитесь, что Google Таблица открыта для чтения (хотя бы по ссылке)
2. **Квоты Google:** Apps Script имеет лимиты (~100 запросов в минуту), но для викторины этого достаточно
3. **CORS:** Веб-приложение должно быть опубликовано с доступом "Все пользователи"
4. **HTTPS:** Викторина должна работать на HTTPS (или localhost для тестов)
